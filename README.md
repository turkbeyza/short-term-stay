# 🤖 Stay AI Agent: Smart Short-Term Stay

Search, book, and manage your stays entirely through a modern AI chat interface or via universal **Model Context Protocol (MCP)** tools.


- **Presentation Video**: https://youtu.be/9ygivog-vgg


## 🌟 Key Features

- **Intelligence-First Search**: The "Stay AI Agent" understands natural language. Ask for "a place for 2 in Izmir next July" and it handles the filters, date conversions, and availability checks automatically.
- **Seamless Booking**: Book your stay directly through the chat. The agent handles guest registration and reservation logic.
- **Rich Chat Experience**: A modern Next.js 16 frontend with markdown support, real-time listing cards, and smooth animations.



## 🏗️ Technical Highlights

- **Agentic Loop**: The AI agent uses a multi-step "think → call tool → analyze result → respond" loop to ensure accurate and grounded answers.
- **Stateless MCP**: The HTTP/SSE MCP server is designed to be stateless for high scalability and easy integration.
